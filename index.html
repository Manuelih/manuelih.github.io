Minecraft Development

Advanced Backend Plugins

Sviluppo di plugin enterprise-grade per Paper, Spigot e Velocity (Proxy). Implementazione di sistemi distribuiti tramite Redis Pub/Sub, gestione persistenza dati ultra-veloce con pooling SQL (HikariCP) e ottimizzazione delle performance tramite profilazione dei thread (Spark/Timings). Utilizzo di NMS (Net.Minecraft.Server) per manipolazione avanzata di entità e pacchetti senza degradare il TPS.

Java (NMS) Redis HikariCP

Low-Level Modding (Forge/Fabric)

Creazione di mod tecniche a basso livello utilizzando Mixins per l'injection di bytecode direttamente nella logica di gioco. Sviluppo di motori di rendering custom per item 3D, ottimizzazione delle pipeline di calcolo sul lato server e integrazione di meccaniche cross-platform. Esperienza consolidata nella risoluzione di conflitti tra mod e nel tuning del garbage collector per server ad alto carico.
